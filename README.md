# 企业IT资产管理系统 python1151

## 项目概述

企业IT资产管理系统是一个基于Python语言的综合性资产管理平台，使用Django框架进行后端开发，Vue.js进行前端开发，旨在帮助企业管理IT资产，提高工作效率。

## 技术栈

- **后端：**
  - Python 3.7
  - Django
  - MySQL
- **前端：**
  - Vue.js
- **开发工具：**
  - PyCharm
